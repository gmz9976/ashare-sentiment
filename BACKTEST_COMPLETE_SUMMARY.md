## 🛠️ 使用方法

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 运行回测
```bash
# 基础回测
ashare-sentiment backtest backtest_config.yaml \
  --start-date 2020-01-01 \
  --end-date 2024-12-31 \
  --summary

# 自定义参数回测
ashare-sentiment backtest backtest_config.yaml \
  --start-date 2020-01-01 \
  --end-date 2024-12-31 \
  --strategies ice_point momentum \
  --benchmarks sh000001 sh000300 sh000905 \
  --initial-capital 1000000 \
  --summary
```

### 3. 查看结果
- HTML报告：`{output_dir}/backtest_report_{timestamp}.html`
- JSON数据：`{output_dir}/backtest_data_{timestamp}.json`
- 图表文件：6个PNG格式的专业图表
