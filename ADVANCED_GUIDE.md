# 高级情绪分析指南

## 🚀 快速体验高级功能

### 1. 下载数据并运行高级分析

```bash

ashare-sentiment download --source akshare --start-date 2024-11-01 --end-date 2024-11-30 --max-stocks 50 --delay 1.0

# 运行高级分析
ashare-sentiment analyze example_config_download.yaml --output advanced_sentiment.csv --advanced --analysis
```

## 📊 输出文件说明

### 1. sentiment.csv - 情绪分析结果
包含以下列：
- `trade_date`: 交易日期
- `sentiment_score`: 情绪得分（-3到+3）
- `sentiment_state`: 情绪状态（主升/分歧/修复/退潮/冰点）
- `is_ice_point`: 是否为冰点
- `warming_signal`: 转暖信号

### 2. analysis_report_*.json - 详细分析报告
包含：
- 当前市场状态
- 关键指标数值
- 投资建议
- 冰点和转暖信号

## 🎯 情绪状态解读

### 主升浪 (MAIN_RISE)
- **特征**: 上涨比例>70%，涨停多，量能放大
- **策略**: 可适当参与强势股，注意风险控制
- **信号**: 市场情绪高涨，赚钱效应明显

### 弱分歧 (WEAK_DIVERGENCE)
- **特征**: 涨跌相对平衡，分歧较小
- **策略**: 可适度参与，关注板块轮动
- **信号**: 市场相对稳定，有结构性机会

### 强分歧 (STRONG_DIVERGENCE)
- **特征**: 涨跌分化明显，破板率高
- **策略**: 谨慎操作，关注龙头股表现
- **信号**: 市场分歧加大，操作难度增加

### 弱修复 (WEAK_RECOVERY)
- **特征**: 小幅反弹，修复力度有限
- **策略**: 谨慎乐观，关注优质标的
- **信号**: 市场开始修复，但力度不强

### 强修复 (STRONG_RECOVERY)
- **特征**: 快速反弹，量能配合
- **策略**: 可积极参与，关注超跌反弹
- **信号**: 市场快速修复，机会较多

### 退潮 (RETREAT)
- **特征**: 连续下跌，涨停减少
- **策略**: 建议观望，等待更好时机
- **信号**: 市场情绪低迷，风险较大

### 冰点 (ICE_POINT)
- **特征**: 极度悲观，成交萎缩
- **策略**: 准备抄底机会，关注转暖信号
- **信号**: 市场极度悲观，可能是底部

## 🔥 冰点识别和转暖信号

### 冰点识别条件
- 上涨比例 < 30%
- 涨停比例 < 1%
- 跌停比例 > 2%
- 成交量萎缩 > 10%

### 转暖信号条件
- 上涨比例比前一日增加 > 10%
- 涨停比例比前一日增加 > 0.5%
- 成交量开始放大

## 📈 关键指标解读

### 情绪强度指标
- **advance_ratio**: 上涨比例，>0.7为强势
- **limit_up_ratio**: 涨停比例，>0.05为强势
- **limit_net_ratio**: 涨跌停净比，>0.03为强势

### 市场活跃度指标
- **volume_change_ratio**: 量能变化，>0.2为放量
- **continuation_stocks**: 连板股票数，>10为强势
- **heaven_earth_count**: 地天板数量，>0为极端反转

### 市场分歧指标
- **break_board_ratio**: 破板率，>0.3为分歧较大
- **seal_board_ratio**: 封板率，>0.7为一致性较强

## 💡 投资建议解读

系统会根据当前市场状态自动生成投资建议：

### 主升浪建议
- 市场情绪高涨，可适当参与强势股
- 注意风险控制，避免追高

### 分歧状态建议
- 市场分歧较大，谨慎操作
- 关注龙头股表现

### 修复状态建议
- 市场快速修复，可积极参与
- 关注超跌反弹机会

### 冰点状态建议
- 市场处于冰点，准备抄底机会
- 关注转暖信号

## 🛠️ 自定义配置

### 调整特征权重
编辑配置文件中的weights部分：

```yaml
weights:
  # 增加涨停权重
  limit_up_ratio: 2.0
  # 减少成交量权重
  volume_change_ratio: 0.5
```

### 调整情绪状态阈值
修改`src/sentiment_ashare/features/sentiment_classifier.py`中的判断条件。

## 📊 实际应用案例

### 案例1: 识别抄底机会
```bash
# 当检测到冰点信号时
if sentiment_state == "冰点" and warming_signal:
    print("准备抄底机会！")
    # 可以开始关注优质标的
```

### 案例2: 风险控制
```bash
# 当检测到强分歧时
if sentiment_state == "强分歧":
    print("市场分歧较大，谨慎操作")
    # 减少仓位，关注龙头股
```

### 案例3: 趋势跟踪
```bash
# 当检测到主升浪时
if sentiment_state == "主升":
    print("市场情绪高涨，可适当参与")
    # 可以增加仓位，关注强势股
```
