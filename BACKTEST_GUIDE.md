# A股情绪回测系统使用指南

## 概述

A股情绪回测系统是一个完整的量化回测框架，用于验证情绪指标的有效性并与上证、深证、中证指数进行对比分析。

## 功能特点

- **多策略支持**：冰点策略、情绪动量策略、逆向策略
- **全面绩效分析**：夏普比率、最大回撤、相关性分析等
- **情绪指标验证**：冰点信号、转暖信号、情绪状态分析
- **可视化报告**：HTML报告、图表分析、JSON数据导出
- **指数对比**：与上证、深证、沪深300、中证500等主要指数对比

## 快速开始

### 1. 安装依赖

```bash
pip install -e .
```

### 2. 运行回测

```bash
# 基础回测
ashare-sentiment backtest backtest_config.yaml \
  --start-date 2020-01-01 \
  --end-date 2024-12-31 \
  --summary

# 自定义策略和基准
ashare-sentiment backtest backtest_config.yaml \
  --start-date 2020-01-01 \
  --end-date 2024-12-31 \
  --strategies ice_point momentum \
  --benchmarks sh000001 sh000300 sh000905 \
  --initial-capital 1000000 \
  --summary
```

### 3. 查看结果

回测完成后会生成详细的HTML报告，包含：
- 策略绩效对比表格
- 累计收益率图表
- 回撤分析图表
- 情绪指标有效性分析
- 冰点信号效果图
- 情绪状态分布图

## 策略说明

### 1. 冰点策略 (ice_point)
- **策略逻辑**：在市场冰点时买入，转暖信号出现时卖出
- **参数配置**：
  - `holding_days`: 最大持有天数（默认10天）
  - `stop_loss`: 止损比例（默认5%）
  - `min_ice_point_score`: 最小冰点得分阈值（默认-2.0）

### 2. 情绪动量策略 (momentum)
- **策略逻辑**：基于情绪得分的动量变化进行交易
- **参数配置**：
  - `lookback_days`: 动量计算回看天数（默认5天）
  - `threshold`: 动量阈值（默认0.5）
  - `min_sentiment_score`: 最小情绪得分阈值（默认-1.0）
  - `max_sentiment_score`: 最大情绪得分阈值（默认1.0）

### 3. 逆向策略 (contrarian)
- **策略逻辑**：在市场极度悲观时买入，极度乐观时卖出
- **参数配置**：
  - `extreme_low_threshold`: 极端悲观阈值（默认-2.5）
  - `extreme_high_threshold`: 极端乐观阈值（默认2.5）
  - `holding_days`: 最大持有天数（默认15天）

## 配置文件说明

### backtest_config.yaml

```yaml
# 数据源配置
provider:
  type: download
  download:
    source: akshare              # 数据源
    start_date: "2020-01-01"     # 开始日期
    end_date: "2024-12-31"       # 结束日期

# 特征权重配置
weights:
  advance_decline: 1.0         # 涨跌比权重
  limit_up_down: 1.0           # 涨跌停净比权重
  # ... 其他权重配置

# 回测配置
backtest:
  strategies:
    - ice_point                # 要测试的策略
    - momentum
    - contrarian
  
  benchmarks:
    - sh000001                 # 基准指数
    - sh000300
    - sh000905
  
  strategy_params:             # 策略参数
    ice_point:
      holding_days: 10
      stop_loss: 0.05
```

## 命令行参数

### 基础参数
- `--start-date`: 回测开始日期（必需）
- `--end-date`: 回测结束日期（必需）
- `--output-dir`: 报告输出目录（默认：./backtest_reports）

### 策略参数
- `--strategies`: 要测试的策略列表（默认：ice_point momentum）
- `--benchmarks`: 基准指数列表（默认：sh000001 sh000300 sh000905）

### 回测参数
- `--initial-capital`: 初始资金（默认：1000000）
- `--transaction-cost`: 交易成本比例（默认：0.001）
- `--advanced`: 使用高级特征（默认：True）

### 输出参数
- `--summary`: 打印回测摘要

## 输出结果

### 1. HTML报告
- 位置：`{output_dir}/backtest_report.html`
- 内容：完整的回测分析报告，包含图表和表格

### 2. JSON数据
- 位置：`{output_dir}/backtest_data_{timestamp}.json`
- 内容：原始回测数据，可用于进一步分析

### 3. 图表文件
- `cumulative_returns.png`: 累计收益率对比图
- `drawdowns.png`: 回撤对比图
- `sentiment_correlation.png`: 情绪相关性分析图
- `ice_point_signals.png`: 冰点信号效果图
- `sentiment_state_distribution.png`: 情绪状态分布图
- `performance_radar.png`: 绩效雷达图

## 绩效指标说明

### 基础指标
- **总收益率**: 整个回测期间的总收益率
- **年化收益率**: 年化后的收益率
- **年化波动率**: 年化后的波动率
- **夏普比率**: 风险调整后收益率
- **索提诺比率**: 下行风险调整后收益率
- **最大回撤**: 最大回撤幅度
- **卡尔玛比率**: 年化收益率/最大回撤
- **胜率**: 正收益交易日比例
- **盈亏比**: 平均盈利/平均亏损

### 情绪指标有效性
- **情绪-市场相关性**: 情绪得分与市场收益率的相关性
- **冰点信号准确性**: 冰点信号后各期间的表现
- **转暖信号及时性**: 转暖信号后各期间的表现
- **情绪状态分析**: 不同情绪状态下的市场表现

## 使用建议

### 1. 数据准备
- 确保有足够的历史数据（建议至少2年）
- 使用akshare作为免费数据源
- 合理设置下载参数避免被拉黑

### 2. 策略选择
- 根据市场特点选择合适的策略
- 可以同时测试多个策略进行对比
- 注意策略参数的敏感性分析

### 3. 风险控制
- 设置合理的初始资金和交易成本
- 关注最大回撤和夏普比率
- 考虑不同市场环境下的策略表现

### 4. 结果解读
- 重点关注情绪指标的有效性
- 对比不同策略的相对表现
- 分析冰点和转暖信号的准确性

## 注意事项

1. **数据质量**: 回测结果依赖于数据质量，建议使用可靠的数据源
2. **参数调优**: 策略参数对结果影响较大，建议进行参数敏感性分析
3. **市场环境**: 不同市场环境下策略表现可能差异较大
4. **过拟合风险**: 避免过度优化参数导致过拟合
5. **实盘差异**: 回测结果与实际交易可能存在差异，注意交易成本、滑点等因素

## 故障排除

### 常见问题

1. **数据下载失败**
   - 检查网络连接
   - 增加请求延迟参数
   - 减少下载股票数量

2. **内存不足**
   - 减少回测时间范围
   - 降低股票数量
   - 使用更少的数据特征

3. **图表显示异常**
   - 确保安装了matplotlib和seaborn
   - 检查中文字体支持
   - 验证数据完整性

### 技术支持

如遇到问题，请检查：
1. 依赖库是否正确安装
2. 配置文件格式是否正确
3. 数据是否完整下载
4. 系统资源是否充足
